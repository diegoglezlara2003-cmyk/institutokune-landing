# Garabato by Kune — Landing

Página principal de **Instituto Kune A.C.** (institutokune.org). Estática, sin
build: `index.html` se abre directo o se sirve con GitHub Pages.

- **Probar Demo** → la app en app.institutokune.org
- **Solicitar Piloto / contacto** → formulario + contacto@institutokune.org
- `aviso-de-privacidad.html` — aviso de privacidad
- `assets/` — ilustraciones de crayón (flecha, amapola, ciudad)

Dependencias por CDN (Tailwind, Three.js, GSAP) — requiere conexión.
